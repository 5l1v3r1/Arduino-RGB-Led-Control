# Arduino - RGB Led Control

Arduino ile yapılan bu projede, RGB Ledler sayesinde saniyede bir
ışık rengi değiştirmektedir.

# Kullanılan Malzemeler :

1 Adet Arduino

1 Adet BreadBoard

1 Adet RGB Led

3 Adet Direnç

7 Adet iki ucu erkek jumper kablo
